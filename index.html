Naive Bayes

El algoritmo de Naive Bayes es un método de clasificación basado en el teorema de Bayes, que asume que las características son independientes entre sí dadas las clases.

Naive Bayes se utiliza para clasificar datos en diferentes categorías basándose en la probabilidad de que un nuevo dato pertenezca a cada clase.

Condiciones de uso:

  • Las características son independientes entre si
  • Las características siguen una distribucion Normal
  • Suficiencia de datos para poder entrenar el modelo
